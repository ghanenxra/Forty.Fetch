# FortyFetch

FortyFetch is a Windows YouTube downloader with a modern dark UI and cyan accent design.

## Features
- YouTube video download support
- Quality options from 360p to 8K (up to 60fps when available)
- MP3 audio extraction
- Custom save folder picker
- Progress, speed, and status display
- Built with `customtkinter`, `yt-dlp`, and bundled `ffmpeg/ffprobe`

## Tech Stack
- Python
- customtkinter
- yt-dlp
- FFmpeg
