# Forty.Fetch
FortyFetch is a premium, high-performance media extraction tool built with Python. It leverages yt-dlp for powerful downloading capabilities and CustomTkinter for a sleek, modern, and high-DPI aware user interface. Designed for speed and reliability, it features automatic quality fallback and prioritizes 60FPS video streams.
